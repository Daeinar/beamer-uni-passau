####Usage
Copy `beamerthemeunipassau.sty`  and `logo.pdf` to the folder containing your main slide file. Then include the style by adding
```
\mode<presentation>
{
  \usetheme{unipassau}
  \setbeamercovered{transparent}
}
```
to your main slide file.
